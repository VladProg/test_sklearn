Американська компанія Microsoft із 10 січня припиняє підтримку операційної системи Windows 8.1.

Про це повідомляє Укрінформ із посиланням на блог компанії.
«10 січня 2023 року підтримка Windows 8.1 закінчиться, після чого технічна допомога та оновлення програмного забезпечення більше не надаватимуться», - йдеться у повідомленні.
Компанія рекомендує оновити пристрої під керуванням Windows 8.1 до більш актуального та підтримуваного випуску Windows.
Якщо пристрій не відповідає технічним вимогам для запуску поточного випуску Windows, рекомендується замінити його на пристрій, що підтримує Windows 11, додали в компанії.
"Корпорація Microsoft не пропонуватиме програму розширеного оновлення безпеки (ESU) для Windows 8.1. Продовження використання Windows 8.1 після 10 січня 2023 року може збільшити схильність організації до ризиків безпеки або вплинути на її здатність виконувати зобов'язання за відповідністю», - йдеться у заяві.
Як повідомлялося, Microsoft та HP передали українським педагогам майже три тисячі ноутбуків.
